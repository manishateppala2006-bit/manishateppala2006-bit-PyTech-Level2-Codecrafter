Library Management System (CRUD)

A Python-based Library Management System that implements CRUD operations:

Add Book – Add new books dynamically.

Issue Book – Issue books to users if available.

Return Book – Return issued books.

Delete Book – Remove books from the library.

View Available Books – List all currently available books.

Dataset: Stored in data.txt (user-generated). Each entry includes Book ID | Title | Status with status available or issued.

Features:

Modular Python code (model.py for logic, main.py for interface, utils.py for helpers).

Input validation to prevent crashes.

Only built-in Python libraries used.

How to Run:

Clone the repo.

Ensure Python 3.x is installed.

Run python main.py and follow the menu options.

Changes are automatically saved to data.txt.
